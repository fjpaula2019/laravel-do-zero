Iniciar o git - git init
Verificar se os arquivos estão na pasta - git status
Adicionando os arquivos para envio = git add .
Colocar o nome do commit - git commit -m "Git Hub com Laravel"
Redirecionar para  a pasta no git - git remote add origin https://github.com/fjpaula2019/laravel-do-zero.git
Subir aqruivo - git push origin master  
Colocar login e senha
